How to Play:

  • Fill the 9x9 grid so that each row, column, and 3x3 box contains digits 1-9
  • Click on a cell and type a number (1-9) to fill it
  • Pre-filled cells (darker background) cannot be changed
  • Use "Hint" to get help with one cell
  • Use "Check Solution" to verify your answers
  • Use "Clear" to remove all your entries
  • Use "Reset" to start over with the same puzzle
